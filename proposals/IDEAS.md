# Ideas

A scratchpad of things the community has been kicking around. Nothing
here is a promise — ideas without champions stay ideas. If one of these
grabs you, pick it up: open a formal RFP in `/proposals/` and champion
it through. See [`CONTRIBUTING.md`](../CONTRIBUTING.md) for how. When
someone champions an idea, update its status to `in progress — <handle>`
or remove the idea entirely once the proposal lands.

---

### NWA builder directory

A curated, searchable list of local agent builders and what they're
working on. Could be generated straight from `community/members/`
profiles — maybe a static site, maybe just a rolled-up index page.
Useful for finding collaborators, useful for anyone new to the scene
trying to figure out who's doing what.
**Status**: open for a champion

### Shared eval harness

A common harness for running the same prompts and scoring across
Claude Code, Codex, Cursor, OpenClaw. Everyone's running private evals
in parallel; a shared one makes us collectively smarter about what
actually works. `proposals/0000-example.md` is a worked example
sketching this out — surfacing it here so someone can champion it
formally.
**Status**: open for a champion

### Local meetup scraper agent

An agent that crawls meetup.com, Eventbrite, Luma, and local event feeds
across NWA to find AI / builder / developer events, then posts a weekly
or monthly digest somewhere the community watches. Low-stakes first
agent project with a real output the group would actually use.
**Status**: open for a champion
